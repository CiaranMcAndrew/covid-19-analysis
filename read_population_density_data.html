Setup

Download the following:
  1. National Identifier Grid https://sedac.ciesin.columbia.edu/data/set/gpw-v4-national-identifier-grid-rev11/data-download
  2. Gridded Population of the World https://beta.sedac.ciesin.columbia.edu/data/set/gpw-v4-population-density/data-download

Usage

The data for population density can be accessed through the API. Currently this consists of the following:
  • Nation - a class to represent the data related to a single nation

Example:

name = 'United Kingdom of Great Britain and Northern Ireland';
nation = Nation.FindNation("NAME0",name )
nation =
